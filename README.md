# Vitol | 60 Years Timeline

A premium, high-energy interactive timeline celebrating 60 years of Vitol's journey.

## Features
- **Interactive GSAP Timeline**: Scroll-triggered path drawing and event reveals.
- **Premium Aesthetics**: Dark mode with radial gradients, flowing energy fields, and tech-circuit background elements.
- **Advanced Animations**: Data packet flow, scanner beams, and halo pulses on milestone nodes.
- **Live Countdown**: Real-time ticker counting down to the 60th-anniversary celebration on April 30, 2026.
- **Fully Responsive**: Optimized for all screen sizes and mobile-ready for iframe embedding.

## Deployment
This project is configured for one-click deployment on **Vercel**.
